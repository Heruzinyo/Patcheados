**Estou com dificuldades para instalar o patch…**

Veja esse vídeo [tutorial de como instalar o patch do Rewrite](https://youtu.be/Tq40xKpZFPs).

**Minha fonte está sobreposta, o que eu faço?**

Pegue esta fonte [AQUI](https://drive.google.com/uc?id=1PyCbHexus84IipG4bQjQ088VT4YhKOmw&export=download) e instale-a no Windows (copie para C:\Windows\Fonts).

**Não consigo abrir, está travando na hora de executar!**

Se depois de instalar o patch, o executável `SiglusEngine.exe` travar, tente substituí-lo com este `.exe` [AQUI](https://drive.google.com/uc?id=1bkYzv-V4toXgmuup_o6aXhYczzZ2GOzj&export=download). Se ainda continuar travando, tente este [AQUI](https://drive.google.com/uc?id=1nKrUvroFan78dEEVJPlwaqmSrvDlxJwm&export=download).
